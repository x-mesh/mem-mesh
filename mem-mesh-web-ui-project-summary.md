## 🔧 해결된 주요 기술 이슈

### 1. sqlite-vec 확장 로딩 문제
**문제**: macOS Python의 기본 SQLite가 확장 로딩을 지원하지 않음  
**해결**: `pysqlite3` 패키지 설치 및 우선 사용 설정  
**결과**: Vector 검색 기능 완전 구현, 실제 유사도 점수 제공

### 2. API 응답 구조 불일치
**문제**: 프론트엔드에서 `searchResult.memories` vs `searchResult.results` 혼용  
**해결**: 모든 페이지에서 일관된 `searchResult.results` 사용  
**결과**: 모든 API 호출 정상화

### 3. 앱 초기화 타이밍 이슈
**문제**: 페이지 로드 시 `window.app`이 준비되지 않아 오류 발생  
**해결**: 앱 준비 대기 로직 + 직접 API 호출 fallback 구현  
**결과**: 안정적인 페이지 로딩 및 데이터 표시

### 4. Memory Detail 페이지 행 문제
**문제**: 복잡한 context-timeline 컴포넌트로 인한 무한 로딩  
**해결**: 단순한 리스트 기반 컨텍스트 표시로 교체  
**결과**: 빠르고 안정적인 메모리 상세 페이지
